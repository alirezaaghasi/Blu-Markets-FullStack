# Blu Markets v5

Portfolio management prototype with three-layer allocation (Foundation/Growth/Upside).

## Features (v5)
- ✅ Loan Repayment Flow
- ✅ Reset Confirmation Modal
- ✅ Rebalance Preserves Frozen Collateral
- ✅ Right Panel Onboarding Experience
- ✅ Compound Validation (Foundation Minimum Check)

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Tech Stack
- React 18
- Vite 5
- Single-file component (no external CSS)
